# azurefs
azurefs is tool for list, upload, download files from azure storage.

# Usage

```
Usage:
  azurefs [command]

Available Commands:
  download    Down load files from azure storage
  help        Help about any command
  list        List files of azure storage
  upload      Upload files to azure storage

Flags:
  -a, --account string     Azure Storage Acount Name. 
      --config string      config file (default is $HOME/.azurefs.yaml)
  -c, --container string   Azure Storage Acount Container Name. (default "share")
  -h, --help               help for azurefs

Use "azurefs [command] --help" for more information about a command.
```
